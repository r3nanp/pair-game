# 🤔 How to run

Clone repository:
```bash
git clone https://github.com/r3nanp/pair-game.git
```

At the root of the project, run:
```bash
yarn
```
or

```sh
npm install
```

Run the following command in order to start the application in a development environment:
```bash
yarn dev
```
or

```bash
npm run dev
```
